# aws-serverless-email-sentiment-inbox
A serverless smart inbox built on AWS that analyzes email sentiment in real time and automatically routes, escalates, or responds based on urgency and tone using Amazon Comprehend.
This is a demonstration of the full potential of the intelligent email inbox in the serverless world. All the relevant services that have been depicted in the following flow are offered by Amazon AWS. Received mail goes to Amazon S3, Amazon Lambda, Amazon Comprehend, and Amazon CloudWatch. This is the best solution for the automated support email inbox.
